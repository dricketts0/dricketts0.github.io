About Me

I am embarking on a new career path as a web developer. It was a path I did not expect to take. The first time I wrote code was more like a dare to myself, to see if I could do it. Then I discovered the awe of creating something out of nothing. It still amazes me to witness an application as an entity that interacts with people in real space and time. It's almost as if it has life on its own, and it all began with a blank screen.

I love creative problem-solving. As a beginner, I make plenty of problems for myself to solve, but each time I'm debugging lines of code, I feel like I'm learning valuable lessons. Each a gift. As my skills develop with experience, I'm taking on pride as I build tools for others to use, making their lives easier. I have found myself wanting to build web applications with the ambition of becoming fully versed in front-end and back-end web development.
